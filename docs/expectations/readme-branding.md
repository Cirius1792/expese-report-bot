# README Branding Expectations

## Happy path

- The project logo is stored at `docs/assets/logo.png`.
- The README renders the logo from that relative path at 240 pixels wide with alternative text `(ex)SpenserBot logo`.
- The visible README heading is `(ex)SpenserBot`.
- The opening description identifies the product as `(ex)SpenserBot`.

## Behaviors that must not happen

- The temporary root-level `logo.png` must not remain.
- Technical identifiers such as `expense-report-bot`, `expense-bot`, `expense-extract`, and `expense_report` must not be renamed as part of this branding-only change.
- Unrelated files must not be modified.
